To finalize the installation and get started with sensenet ECM Preview, please follow these steps:

1. Build your solution, make sure that there are no build errors.

2. Install sensenet ECM Preview using SnAdmin

   - open a command line and go to the \Admin\bin folder
   - execute the install-preview command with the SnAdmin tool

   .\snadmin install-preview
   

For more information and support, please visit http://sensenet.com
